void arl::mujoco::Robot::waitNextCycle ( )
virtual

Waits for the next cycle of the control loop.

In the case of a MuJoCo Robot it reades the simulation time and moves MuJoCo simulation one time step ahead. Use it at each control loop as in real robots.
